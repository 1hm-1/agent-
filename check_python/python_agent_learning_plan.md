# Python 学习清单：面向 Agent 开发的复习路线

适合对象：很久没碰过 Python，希望重新建立基础，并逐步过渡到 Agent 开发。

建议节奏：`17` 节课，适合用 `3` 到 `6` 周完成。

使用方式：

- 每节课只追求“看懂 + 能写一点 + 能跑起来”
- 不要求一次学很深，先恢复手感
- 学完基础后，再进入 API、异步、项目结构和 Agent 实战

---

## 第一阶段：找回 Python 手感

### 第 1 课：重新认识 Python 与开发环境

- Python 安装与运行方式
- 脚本、解释器、编辑器的关系
- 第一个 Python 程序
- 注释、缩进、基本代码规范

### 第 2 课：变量、基本数据类型与输入输出

- 变量
- 字符串、整数、浮点数、布尔值
- 类型转换
- `print()` 与 `input()`

### 第 3 课：条件判断与流程控制

- `if / elif / else`
- 比较运算符
- 逻辑运算符
- 常见判断写法

### 第 4 课：列表、元组、集合、字典

- 列表的增删改查
- 元组的特点
- 集合的去重用途
- 字典的键值结构

### 第 5 课：循环与遍历

- `for` 循环
- `while` 循环
- `break` 与 `continue`
- 遍历列表和字典

---

## 第二阶段：写出可复用的 Python 代码

### 第 6 课：函数基础

- 定义函数
- 参数与返回值
- 默认参数
- 位置参数与关键字参数

### 第 7 课：常见内置函数与数据处理

- `len()`、`range()`、`enumerate()`
- `sorted()`、`sum()`、`max()`、`min()`
- 字符串常见方法
- 列表推导式入门

### 第 8 课：异常处理与调试基础

- 什么是异常
- `try / except / finally`
- 常见报错类型
- 看懂报错信息与定位问题

### 第 9 课：模块、包与代码拆分

- `import` 的基本用法
- 标准库与第三方库
- 一个 Python 文件就是一个模块
- 如何拆分小项目文件

---

## 第三阶段：进入真实开发必备能力

### 第 10 课：文件读写与路径处理

- 读写文本文件
- `with open(...)`
- 编码与 UTF-8
- 基础路径处理

### 第 11 课：JSON、配置与数据交换

- JSON 基本结构
- `json.load` 与 `json.dump`
- Python 字典和 JSON 的关系
- 配置数据的组织方式

### 第 12 课：面向对象基础

- 类与对象
- `__init__`
- 实例属性与方法
- 什么时候该用类

### 第 13 课：环境变量与项目配置管理

- 环境变量的概念
- 读取 API Key
- 为什么不能把密钥写死
- 配置与代码分离

---

## 第四阶段：面向 Agent 开发的关键 Python 能力

### 第 14 课：HTTP 请求与 API 调用基础

- 请求与响应
- `requests` 的基本用法
- GET 与 POST
- 超时、状态码、错误处理

### 第 15 课：消息结构、上下文与简单聊天程序

- 用列表和字典表示消息
- `role` 与 `content`
- 维护历史上下文
- 写一个最小聊天循环

### 第 16 课：异步编程入门与 Agent 项目过渡

- `async` 与 `await`
- 同步和异步的区别
- 为什么 API 调用常配合异步
- 一个简单 Agent 项目的代码结构

### 第 17 课：`re` 正则表达式基础与返回解析器入门

- `re.search`、`re.match`、`re.findall`
- 分组与 `group()`
- 贪婪与非贪婪匹配
- 用正则解析 `Action: Search[...]`

---

## 学完这 16 课后，你应当具备的能力

- 能读懂基础 Python 项目
- 能写函数、类、模块
- 能处理 JSON、文件、异常
- 能调用外部 API
- 能理解消息列表和上下文维护
- 能开始学习 Agent 框架和工具调用

---

## 建议的后续学习顺序

完成这份清单后，可以继续进入下面这些主题：

1. `requests` 进阶与 API 封装
2. `asyncio` 基础实战
3. `pytest` 测试基础
4. 简单日志系统
5. OpenAI / LLM API 调用
6. Prompt 设计
7. Tool calling
8. 简单 RAG
9. 多 Agent 基础概念
