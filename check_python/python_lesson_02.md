# 第 2 讲：变量、基本数据类型与输入输出

## 这一讲的目标

学完这一讲，你应该能做到：

- 理解什么是变量
- 区分常见基本数据类型
- 知道字符串、整数、浮点数、布尔值分别是什么
- 会使用 `print()` 输出内容
- 会使用 `input()` 接收用户输入
- 理解简单的类型转换

这一讲是后面所有 Python 学习的基础。很多报错、很多逻辑问题，本质上都和“变量里到底装的是什么类型的数据”有关。

---

## 1. 什么是变量？

变量可以理解成“一个有名字的盒子”，你把数据放进去，后面就可以通过这个名字再次使用它。

例如：

```python
name = "Alice"
age = 18
```

这里：

- `name` 是变量名
- `"Alice"` 是变量里保存的值
- `age` 是另一个变量名
- `18` 是它保存的值

在 Python 里，使用等号 `=` 就是在做“赋值”。

注意：

- 这里的 `=` 不是“数学上的相等”
- 它更像“把右边的值交给左边这个变量”

例如：

```python
message = "Hello"
print(message)
```

运行时会输出：

```text
Hello
```

---

## 2. 变量为什么重要？

因为程序本质上就是：

- 接收数据
- 处理数据
- 输出结果

而变量就是用来保存和传递这些数据的。

后面你写 Agent 程序时，经常会把这些信息保存在变量里：

- 用户输入
- 消息列表
- 模型名
- API 返回结果
- 配置项

所以变量不是一个小知识点，而是整个程序的基础。

---

## 3. 变量命名的基本规则

### 3.1 合法命名

变量名通常可以包含：

- 字母
- 数字
- 下划线 `_`

例如：

```python
user_name = "Tom"
age2 = 20
total_count = 5
```

### 3.2 不合法命名

下面这些写法不行：

```python
2age = 18
user-name = "Tom"
class = "test"
```

原因分别是：

- 不能以数字开头
- 不能包含减号
- 不能使用 Python 关键字

### 3.3 命名建议

尽量让变量名有意义。

不推荐：

```python
a = "Tom"
b = 18
```

更推荐：

```python
user_name = "Tom"
user_age = 18
```

这会让代码更容易读懂。

---

## 4. 基本数据类型：字符串

字符串就是文本。

例如：

```python
name = "Alice"
city = "Shanghai"
message = "Hello, world"
```

通常用单引号或双引号包起来：

```python
"hello"
'hello'
```

这两种写法在大多数情况下都可以。

### 字符串的特点

- 用来表示文本
- 可以包含中文、英文、数字、符号
- 本质上还是“文本”，不是数字

例如：

```python
"123"
```

这是字符串，不是整数。

---

## 5. 基本数据类型：整数

整数就是没有小数部分的数字。

例如：

```python
age = 18
year = 2026
count = 100
```

整数适合表示：

- 年龄
- 数量
- 次数
- 编号

---

## 6. 基本数据类型：浮点数

浮点数就是带小数的数字。

例如：

```python
price = 19.9
score = 88.5
temperature = 36.7
```

浮点数适合表示：

- 价格
- 温度
- 平均值
- 比例

---

## 7. 基本数据类型：布尔值

布尔值只有两个：

```python
True
False
```

注意首字母必须大写。

布尔值通常用于表示“是否”：

```python
is_student = True
is_logged_in = False
```

后面你会在条件判断里频繁用到它。

---

## 8. 用 `type()` 看变量类型

如果你不确定一个值是什么类型，可以用：

```python
print(type("hello"))
print(type(18))
print(type(19.9))
print(type(True))
```

你会看到类似输出：

```text
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
```

这里先不用死记这些英文，只要知道：

- `str` 表示字符串
- `int` 表示整数
- `float` 表示浮点数
- `bool` 表示布尔值

---

## 9. `print()`：把内容输出出来

`print()` 是最常用的输出函数。

例如：

```python
print("Hello")
print(123)
```

也可以输出变量：

```python
name = "Alice"
print(name)
```

### 一次输出多个值

```python
name = "Alice"
age = 18
print(name, age)
```

输出大致会是：

```text
Alice 18
```

`print()` 在学习阶段非常重要，因为它是你最直接的调试工具。

当你不确定变量里装了什么时，先 `print()` 出来看看，通常很有帮助。

---

## 10. `input()`：接收用户输入

`input()` 会等待用户在终端输入内容。

例如：

```python
name = input("请输入你的名字：")
print("你好，", name)
```

运行后，程序会停下来等你输入。

### 一个非常重要的点

`input()` 返回的内容默认是字符串。

例如你输入：

```text
18
```

程序拿到的其实是：

```python
"18"
```

而不是整数 `18`。

这个点非常重要，因为很多初学者的报错都来自这里。

---

## 11. 类型转换

有时候我们需要把一种类型转成另一种类型。

常见写法：

```python
age_text = "18"
age = int(age_text)
print(age)
print(type(age))
```

这里：

- `"18"` 是字符串
- `int("18")` 把它转成整数

类似地：

```python
price = float("19.9")
num_text = str(100)
```

分别表示：

- 把字符串转成浮点数
- 把数字转成字符串

---

## 12. 为什么类型转换很重要？

因为不同类型做同样的操作，结果可能完全不同。

看这个例子：

```python
print("2" + "3")
print(2 + 3)
```

输出分别是：

```text
23
5
```

原因是：

- 字符串相加是拼接
- 数字相加才是数学加法

再看一个常见情况：

```python
age = input("请输入年龄：")
print(age + 1)
```

这通常会报错，因为：

- `age` 是字符串
- 字符串不能直接和整数相加

正确写法通常是：

```python
age = int(input("请输入年龄："))
print(age + 1)
```

---

## 13. 一个完整小例子：读取姓名和年龄

```python
name = input("请输入你的名字：")
age = int(input("请输入你的年龄："))

print("你的名字是：", name)
print("明年你将是：", age + 1)
```

这段代码做了什么：

1. 读取名字
2. 读取年龄
3. 把年龄从字符串转成整数
4. 输出结果

这是一个非常典型的“输入 -> 处理 -> 输出”的小程序结构。

## 14. 初学者常见错误

### 错误 1：把字符串当数字用

例如：

```python
num = input("请输入数字：")
print(num + 1)
```

问题在于 `input()` 返回的是字符串。

### 错误 2：变量名起得太随意

例如：

```python
a = "Tom"
b = 18
c = True
```

短期看能跑，长期非常难维护。

### 错误 3：覆盖掉内置函数名

例如：

```python
print = "hello"
```

这样后面再写：

```python
print("test")
```

就会出问题，因为你把 `print` 这个函数名拿去当变量用了。
