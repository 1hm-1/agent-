# LeetCode Hot100 速成工作流

## 推荐工作流

1. 在 LeetCode 网站做题并提交通过
2. 本地只沉淀高价值内容，不复制整道题目
3. 每做完一道典型题，在对应题型目录保存代码
4. 对卡住或做错的题，在 `错题复盘.md` 里记录原因
5. 每天刷题前后各看一次 `模板速记.md`

---

## 目录说明

- `计划.md`：一周速成刷题计划
- `Python刷题语法速查表.md`：Python 常用语法与数据结构速查
- `模板速记.md`：常见算法模板
- `错题复盘.md`：错题总结
- `arrays`：数组、双指针、滑动窗口
- `linked_list`：链表
- `tree`：二叉树
- `dp`：动态规划
- `backtracking`：回溯
- `graph`：图论
- `hash_and_stack`：哈希、栈、单调栈
- `binary_search_heap`：二分、堆

---

## 建议命名方式

```text
题号_题目名.py
```

例如：

```text
1_两数之和.py
3_无重复字符的最长子串.py
206_反转链表.py
```

## 每道题建议沉淀内容

- 最终通过代码
- 题型标签
- 核心思路
- 易错点
- 是否需要二刷
