## 使用方法

- 这里只放题型模板和核心识别信号

---

## 1. 双指针

### 识别信号

- 数组有序
- 从两边向中间逼近
- 原地删除/移动元素

### 模板

```python
left, right = 0, len(nums) - 1
while left < right:
    if condition:
        left += 1
    else:
        right -= 1
```

---

## 2. 快慢指针

### 识别信号

- 原地覆盖
- 去重
- 链表找中点/找环

### 模板

```python
slow = 0
for fast in range(len(nums)):
    if condition:
        nums[slow] = nums[fast]
        slow += 1
```

## 6. 二叉树 DFS

### 识别信号

- 遍历
- 路径
- 子树问题

### 模板

```python
def dfs(root):
    if not root:
        return
    dfs(root.left)
    dfs(root.right)
```

---

## 7. 二叉树 BFS

### 识别信号

- 层序遍历
- 最短步数
- 一层一层处理

### 模板

```python
from collections import deque

q = deque([root])
while q:
    node = q.popleft()
    if node.left:
        q.append(node.left)
    if node.right:
        q.append(node.right)
```

## 8. 回溯

### 识别信号

- 所有组合
- 所有排列
- 所有可行解

### 模板

```python
res = []
path = []

def backtrack(start):
    if 结束条件:
        res.append(path[:])
        return

    for i in range(start, n):
        path.append(nums[i])
        backtrack(i + 1)
        path.pop()
```

## 9. 二分搜索

### 识别信号

- 有序数组
- 单调性
- 查找边界

### 模板

```python
left, right = 0, len(nums) - 1
while left <= right:
    mid = (left + right) // 2
    if nums[mid] == target:
        return mid
    elif nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
```
