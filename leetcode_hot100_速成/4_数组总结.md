#### #238 移动零
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

- 1 暴力解法
```py
for i in range(nums.count(0)):
    nums.remove(0)
    nums.append(0)
```

- 2 双指针
设定双指针slow,fast
slow负责逐步覆盖原数组，将非零元素往前排
[3,1,0,4,0,2]

slow = 0 -> 左指针
```py
for fast in range(len(nums)):
    if nums[fast]!=0:
        nums[slow]=nums[fast]
        slow+=1             #非零元素排到前面
for i in range(slow, len(nums)):
    nums[i]=0               #按原长度补齐0
```
