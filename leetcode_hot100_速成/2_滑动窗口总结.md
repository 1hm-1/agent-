### #3 无重复字符的最长子串   -> 不定长滑动窗口
给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。
- 哈希表 dict（空，记录字符上次出现的位置）
- 左指针i：左边界的前一个位置
- 右指针j: 右边界

```py
对 j for 循环：
    if s[j] in dic：(字符重复出现)
        左指针移动，**考虑到当前字符上次出现的位置可能已在窗口外，左指针不能往回拉，所以不能直接i=dic[s[j]]** 想想该怎么处理
    字符第一次出现，s[j]及其索引加入dic:
    dic[s[j]]=j
    
    因为左开右闭，res的长度为j-i，每次检查res长度，若最长，则更新res值
    res=max(res, j-1)
return
```
